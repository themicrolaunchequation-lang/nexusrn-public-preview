NexusRN v223 — Vercel Ready Complete Package

This ZIP contains the full static NexusRN v223 app prepared for Vercel deployment.

Use:
1. Extract this ZIP.
2. Upload the CONTENTS of this extracted folder to a GitHub repository, then import that repo into Vercel.
3. In Vercel project settings:
   - Framework Preset: Other
   - Build Command: leave empty
   - Output Directory: leave empty or .
4. After deployment, add custom domain:
   nexusrn.healthqualityleader.com
5. In GoDaddy DNS, add the CNAME record that Vercel shows for the subdomain.

Important:
- Do not deploy a ZIP that only contains reports.
- The app root is this folder: it contains index.html, vercel.json, public-demo, pricing, practice, data, assets, and legal pages.
- Do not enable Lemon Squeezy paid checkout until the live URL is visually tested.
