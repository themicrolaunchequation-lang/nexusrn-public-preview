NexusRN GitHub Upload Package

IMPORTANT:
Do NOT upload this ZIP directly to GitHub.
GitHub will not extract it for Vercel.

Correct steps:
1. Extract this ZIP.
2. Open the extracted folder.
3. Press Ctrl + A to select EVERYTHING inside the folder.
4. Drag the selected files and folders into GitHub's "uploading an existing file" page.
5. Wait until GitHub finishes uploading all files.
6. Commit changes with:
   Initial NexusRN v223 public preview deployment

The repository root should show index.html directly.
If GitHub shows a folder first and index.html is inside that folder, it is uploaded incorrectly.
